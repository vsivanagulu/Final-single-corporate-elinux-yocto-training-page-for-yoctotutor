# Corporate Training Images

This folder is intended for storing local images used in the corporate training pages.

## Instructions

1. Place your image files in this folder.
2. Update the corresponding component files to import or reference these images.

## Recommended Images

- **Firmware to Linux Migration Hero**: `firmware-hero.jpg`
  - Recommended size: 800x600 or 1200x900 (4:3 aspect ratio)
  - Update `corporate-trainings/firmware-to-linux-migration.tsx` to use this image.

- **Course Thumbnails**:
  - `yocto-bsp-thumb.jpg`
  - `embedded-linux-thumb.jpg`
  - `advanced-yocto-thumb.jpg`
  - `drivers-thumb.jpg`
  - Update `home.tsx` `coursesData` to use these images if replacing the Unsplash placeholders.
